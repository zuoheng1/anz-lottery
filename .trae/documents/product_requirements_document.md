# ANZ 年会扭蛋抽奖系统 - 产品需求文档 (PRD)

## 1. 项目概述
为 ANZ 组织年会开发一款精致、炫酷的 3D 扭蛋抽奖程序。支持约 15 位员工参与，员工自行点击抽奖，奖品为对应序号。系统需支持数据导入导出，并具备高品质的交互动画。

## 2. 用户流程
1. **数据准备**：管理员导入员工名单（人工录入或 Excel/CSV 导入）。
2. **抽奖过程**：
   - 员工点击“开始抽奖”或点击扭蛋机手柄。
   - 屏幕展示炫酷的 3D 扭蛋机动画（球体滚动、掉落）。
   - 扭蛋打开，展示中奖序号及员工姓名（可选）。
3. **结果处理**：记录中奖结果，支持导出为 Excel 表格用于兑奖。

## 3. 功能需求

### 3.1 数据管理
- **员工导入**：
  - 支持手动输入姓名。
  - 支持 Excel/CSV 文件批量导入（姓名、ID 等）。
  - *注：飞书 MCP 导入通过标准文件格式或 JSON 粘贴实现。*
- **奖品配置**：
  - 设定奖品序号范围（例如 1-15 号）。
  - 对应奖品描述（可选，目前主要是序号）。

### 3.2 抽奖核心 (3D 扭蛋机)
- **视觉效果**：
  - 真实的物理引擎模拟（球体碰撞、滚动）。
  - 扭蛋球材质需有质感（反光、透明度）。
  - 场景需有光影效果，营造高端、喜庆或科技感氛围。
- **交互控制**：
  - “扭动”动作触发动画。
  - 动画过程流畅，配合音效（可选）。
  - 结果展示弹窗：包含粒子特效、光芒等“获得感”强的动画。

### 3.3 结果管理
- **中奖列表**：实时显示已中奖人员和对应序号。
- **结果导出**：一键导出 Excel 文件。
- **重置功能**：支持清空数据重新开始。

## 4. 非功能需求 (UI/UX)
- **视觉风格**：精致、炫酷。建议采用深色背景搭配高亮霓虹或金属质感，突出“年会盛典”氛围。
- **动画交互**：
  - 按钮 Hover：发光、缩放、边框流动等效果。
  - 页面转场：平滑过渡，无生硬刷新。
  - 3D 性能：保证在主流浏览器上 60fps 运行。
- **响应式**：适配桌面端浏览器（年会大屏投屏场景优先）。
